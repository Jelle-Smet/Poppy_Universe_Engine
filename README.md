# Poppy Universe - Recommendation Engine

This repository contains the Recommendation Engine for the Poppy Universe project.

## Project Structure

- `Poppy_Universe_Engine/` — C# project for the recommendation engine
- (future) `frontend/` — Vue + Vite frontend
- (future) `backend/` — Node.js backend

## Setup

1. Open `Poppy_Universe_Engine` in Visual Studio.
2. Build and run the project.

## Notes

- This repository is part of the larger Poppy Universe project.
- Make sure to ignore IDE and temporary files, as defined in `.gitignore`.
